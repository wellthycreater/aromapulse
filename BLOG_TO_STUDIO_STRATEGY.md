## 📍 공방 자동 추천 알고리즘

### **위치 기반 추천**
```javascript
// 사용자의 댓글/메시지에서 지역 추출
const region = extractRegion(userMessage);
// 예: "서울", "강남", "부산" 등

if (region) {
  // API 호출: 해당 지역 공방 검색
  const studios = await fetch(
    `/api/workshop-bookings/search-by-region?region=${region}&type=local_studio`
  );
  
  if (studios.length > 0) {
    response += `
      
      📍 ${region} 지역 추천 공방:
      ${studios.map(s => `
        - ${s.title} (${s.distance_formatted})
          [버튼: 예약하기 → /studio-detail?id=${s.id}]
      `).join('\n')}
    `;
  }
}
```

## 📊 성과 측정 지표 (KPI)

### **전환율 추적**
1. **블로그 → 공방 검색 페이지**: 클릭률
2. **공방 검색 → 상세 페이지**: 전환율
3. **상세 페이지 → 예약 완료**: 예약 전환율
4. **예약 → 실제 방문**: 노쇼율

### **목표 수치**
- 블로그 방문자 → 공방 검색: **5%**
- 공방 검색 → 예약 완료: **20%**
- 예약 → 실제 방문: **80%** (노쇼 20%)
