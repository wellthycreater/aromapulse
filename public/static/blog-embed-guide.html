네이버 블로그 챗봇 임베드 가이드

중요: 네이버 블로그는 외부 iframe 삽입이 제한되어 있습니다. 아래 방법들을 참고하세요.

방법 1 링크 버튼으로 챗봇 연결

블로그 포스트에 버튼 이미지를 삽입하고 챗봇 페이지로 링크를 연결합니다.

📌 삽입할 URL:

https://www.aromapulse.kr/chatbot
1

버튼 이미지 삽입

블로그 편집기에서 "사진 삽입" → 버튼 이미지 업로드

2

링크 연결

이미지 선택 → "링크" 버튼 → 위 URL 입력

3

새 창 열기 설정

"새 창으로 열기" 체크

추천: 버튼 텍스트 예시 - "🤖 AI 상담 시작하기", "💬 실시간 상담", "🌿 무료 상담받기"

방법 2 HTML 위젯 삽입 (티스토리, 워드프레스)

티스토리나 워드프레스 블로그에서 직접 임베드할 수 있습니다.

<!-- 아로마펄스 AI 챗봇 위젯 --> <iframe src="https://www.aromapulse.kr/static/chatbot-widget" style="position: fixed; bottom: 0; right: 0; width: 100%; height: 100%; border: none; z-index: 9999;" frameborder="0" allow="microphone" ></iframe>

참고: 네이버 블로그는 iframe을 지원하지 않습니다.

방법 3 댓글 작성자 자동 챗봇 연결

블로그 댓글 작성 시 자동으로 챗봇 대화를 시작하는 시스템입니다.

🔄 자동 연결 흐름

1
사용자가 블로그 댓글 작성
2
댓글 내용이 자동으로 수집 및 AI 분석
3
B2B/B2C 타입 자동 구분
4
관리자 대시보드에 리드 등록
5
댓글에 자동 답글: "상담 시작하기" 링크 제공

📌 API 연동이 필요한 기능:

  • 네이버 블로그 API 키 필요
  • 정기적으로 댓글 수집 (Cron Job)
  • 자동 답글 작성 기능
