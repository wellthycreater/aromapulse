# 블로그 관리 시스템 빠른 시작 가이드

## 🚀 5분만에 시작하기

### 1️⃣ 접속 URL
**개발 환경**: https://3000-ixw6l6ek5pa4nw2e7gi09-c07dda5e.sandbox.novita.ai

### 2️⃣ 관리자 로그인
1. `/login` 페이지로 이동
2. 관리자 계정 로그인 (또는 `/create-admin`에서 새 계정 생성)
   - 비밀 키: `aromapulse-admin-2025`

### 3️⃣ 블로그 관리 페이지
1. `/admin-products` 페이지로 이동
2. **"블로그 관리"** 탭 클릭 (네 번째 탭)

### 4️⃣ 댓글 수집 테스트
1. URL 입력란에 다음 입력:
   ```
   https://blog.naver.com/aromapulse/223921529276
   ```
2. **"댓글 수집 및 분석"** 버튼 클릭
3. 결과 확인:
   - 총 댓글: 5개
   - 구매 의도: 3개
   - B2C: 3개
   - B2B: 2개
   - 생성된 챗봇 세션: 3개

### 5️⃣ 결과 확인
- 하단에 등록된 게시물 카드 표시
- 통계 정보 확인:
  - 댓글 수
  - 구매 의도 댓글 수
  - B2C/B2B 고객 분포
  - 생성된 챗봇 세션 수

## 🎯 주요 기능

### 자동 AI 분석
- ✅ 감정 분석 (긍정/부정/중립)
- ✅ 사용자 타입 예측 (B2C/B2B)
- ✅ 의도 분석 (구매의도, 문의, B2B문의, 가격문의, 긍정리뷰, 일반댓글)
- ✅ 키워드 추출 (증상, 제품, 향, 용도)

### 챗봇 자동 생성
구매 의도가 감지된 댓글에 대해 자동으로:
1. 챗봇 세션 생성
2. 초기 컨텍스트 메시지 생성
3. 사용자 메시지 기록
4. AI 응답 자동 생성

### 비즈니스 가치
- 🎯 블로그 댓글에서 잠재 고객 자동 발굴
- 💬 구매 의도 고객에게 즉시 챗봇 상담 제공
- 📊 B2C/B2B 고객 분포 실시간 파악
- 🚀 수동 관리 → 자동화로 운영 효율 극대화
