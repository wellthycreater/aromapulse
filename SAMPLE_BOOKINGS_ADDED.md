## 🐛 문제 발생 시

### 예약이 보이지 않는 경우

1. **로그인 확인**
   - 올바른 계정으로 로그인했는지 확인
   - 쿠키가 만료되지 않았는지 확인

2. **브라우저 새로고침**
   - F5 또는 Ctrl+R로 페이지 새로고침
   - 캐시 삭제: Ctrl+Shift+R

3. **API 직접 호출**
   ```bash
   # 브라우저 콘솔에서
   fetch('/api/bookings/my-bookings', {credentials: 'include'})
     .then(r => r.json())
     .then(d => console.log(d))
   ```

4. **데이터베이스 확인**
   ```bash
   npx wrangler d1 execute aromapulse-production --remote \
     --command="SELECT COUNT(*) FROM oneday_class_bookings WHERE user_id=2"
   ```
