=== Cloudflare Console에서 순서대로 실행하세요 ===

📌 중요: 각 쿼리를 하나씩 복사 → Console에 붙여넣기 → Execute 클릭

--- 쿼리 1: is_gift_wrapping 컬럼 추가 ---
ALTER TABLE workshop_quotes ADD COLUMN is_gift_wrapping INTEGER DEFAULT 0;

--- 쿼리 2: 인덱스 생성 ---
CREATE INDEX IF NOT EXISTS idx_workshop_quotes_options ON workshop_quotes(is_workation, is_gift_wrapping);

--- 검증 쿼리 1: 테이블 구조 확인 ---
PRAGMA table_info(workshop_quotes);

--- 검증 쿼리 2: 인덱스 확인 ---
PRAGMA index_list(workshop_quotes);

--- 검증 쿼리 3: 기존 데이터 확인 ---
SELECT COUNT(*) as total_quotes, SUM(CASE WHEN is_gift_wrapping = 0 THEN 1 ELSE 0 END) as without_gift_wrapping, SUM(CASE WHEN is_gift_wrapping = 1 THEN 1 ELSE 0 END) as with_gift_wrapping FROM workshop_quotes;

=== 실행 순서 ===
1. 쿼리 1 복사 → 붙여넣기 → Execute → "Success" 확인
2. 쿼리 2 복사 → 붙여넣기 → Execute → "Success" 확인
3. 검증 쿼리 1 실행 → is_gift_wrapping 컬럼 확인
4. 검증 쿼리 2 실행 → idx_workshop_quotes_options 인덱스 확인
5. 검증 쿼리 3 실행 → 모든 기존 데이터가 is_gift_wrapping=0 확인
